# Equity Portfolio Risk Management & Futures Hedging
Built an end-to-end equity portfolio risk management framework involving portfolio construction, return analytics, CAPM beta estimation, and futures-based hedging using Indian equity data.

A comprehensive portfolio risk management project demonstrating equity portfolio construction, return analytics, CAPM beta estimation, and futures-based hedging using Indian equity market data.

---

# Overview

This project develops a diversified equity portfolio using Indian listed companies and applies financial risk management techniques to measure portfolio performance and hedge market risk.

The analysis integrates portfolio analytics, systematic risk estimation, and futures hedging to evaluate the effectiveness of risk reduction strategies.

---

# Objectives

- Construct a diversified equity portfolio
- Compute daily, weekly and monthly returns
- Estimate systematic risk using CAPM
- Calculate portfolio beta
- Design a futures hedge
- Evaluate hedge effectiveness

---

# Project Workflow

Market Data Collection > Portfolio Construction > Return Analysis > Correlation Analysis > CAPM & Beta Estimation > Futures Hedging > Portfolio Performance Evaluation
